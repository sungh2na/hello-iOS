- 코멘트
    - 복잡한 코드를 Comment를 통해 이해에 도움을 주거나, 코드리뷰 할 수 있게 도와줌
    - Comment는 컴파일러가 컴파일 시 무시하는 문장
    - 단일라인 - // (내용), 커맨드 + /
    - 다중라인 - /* (내용) */
